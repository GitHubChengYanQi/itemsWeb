## 简介

基于icejs开发的前端基础框架模板，不包含登录界面。

提供路由配置，左侧菜单配置，状态管理。

## 安装


## 目录结构

```
.
├── README.md       // 模版文档
├── package.json
└── src        // 模版目录
    ├── components       // 组件
    ├── Config       // 接口，菜单，路由等配置
    ├── layouts   // 框架结构
    └── pages    // 页面
```



## 使用

- 启动调试服务: `npm start`
- 构建 dist: `npm run build`
